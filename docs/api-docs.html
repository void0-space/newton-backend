üìñ Overview

Newton WhatsApp API allows you to send messages programmatically using simple HTTP requests. All requests are handled through URL parameters for maximum simplicity.

Base URL

https://api.whatsappgateway.in/api/v1/whatsapp

Protocol

All communication is done over HTTPS. Both GET and POST methods are supported with identical functionality.

üîê Authentication

All requests require an API key. Provide it as a URL parameter:

?apiKey=your-api-key-here

Example:

https://api.whatsappgateway.in/api/v1/whatsapp/send?apiKey=YOUR_API_KEY&to=917601940546&message=Hello

Tip: You can also use the X-API-Key header instead of URL parameter if you prefer.

‚úâÔ∏è Send Message Endpoint

Request

GET POST /send

Parameters

Parameter Type Status Description
apiKey string Required Your API key for authentication
to string Required Recipient's phone number (e.g., 917601940546)
message string Required for text Message content (required for text messages)
type string Optional Message type (text, image, video, audio, document). Default: text
mediaUrl string Required for media URL of media file (required for non-text messages)
fileName string Optional File name for document messages
caption string Optional Caption for media messages
replyMessageId string Optional Message ID to reply to

Response (Success)

{
  "success": true,
  "messageId": "A5C8232A8C65858FAB1A64E140D8C68B",
  "status": "sent",
  "to": "917601940546@s.whatsapp.net",
  "sessionId": "session-id-here",
  "timestamp": "2025-10-27T17:38:59.000Z"
}

üí¨ Message Types

Text Message

Send a simple text message.

Parameters:

  • to - Recipient phone number
  • message - Text content
  • type - "text"

Image Message

Send an image with optional caption.

Parameters:

  • to - Recipient phone number
  • type - "image"
  • mediaUrl - Image URL
  • caption - (Optional) Image caption

Video Message

Send a video with optional caption.

Parameters:

  • to - Recipient phone number
  • type - "video"
  • mediaUrl - Video URL
  • caption - (Optional) Video caption

Audio Message

Send an audio file.

Parameters:

  • to - Recipient phone number
  • type - "audio"
  • mediaUrl - Audio URL

Document Message

Send a document (PDF, DOC, etc.) with optional caption.

Parameters:

  • to - Recipient phone number
  • type - "document"
  • mediaUrl - Document URL
  • fileName - File name
  • caption - (Optional) Document caption

Python Example

import requests

url = "https://api.whatsappgateway.in/api/v1/whatsapp/send"

    "apiKey": "YOUR_API_KEY",
    "to": "917601940546",
    "message": "Hello",
    "type": "text"
}

response = requests.get(url, params=params)
print(response.json())

üí° Best Practices

  • URL Encoding: Always properly URL-encode special characters in parameters
  • Rate Limiting: Limit requests to 100 per minute per API key
  • Error Handling: Implement proper error handling and retry logic
  • Message IDs: Save returned messageId for delivery tracking
  • Valid URLs: Ensure media URLs are publicly accessible and correct format
  • Phone Numbers: Use valid 10-digit phone numbers with country code
